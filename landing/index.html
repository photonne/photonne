Características

Todo lo que necesitas para tu biblioteca multimedia

Photonne combina potentes funciones de organización con una interfaz limpia y rápida, todo bajo tu control.

Línea de tiempo

Vista cronológica de toda tu biblioteca. Navega por años, meses y días de forma intuitiva.

Álbumes y carpetas

Álbumes propios, compartidos con otros usuarios y compartibles mediante enlace público. Navegación por carpetas del sistema de archivos.

Mapa geográfico

Visualiza tus fotos y videos por ubicación gracias a los metadatos GPS extraídos del EXIF.

Favoritos

Marca tus mejores fotos y videos como favoritos para acceder a ellos de forma inmediata.

Buscador avanzado

Busca por nombre, fecha, etiquetas y metadatos EXIF. Encuentra cualquier foto en segundos.

Etiquetas

Tags manuales para organizar tu biblioteca. Clasificación automática por ML próximamente.

Archivado y Papelera

Archiva assets para ocultarlos sin eliminarlos. La papelera permite restaurar lo eliminado antes de borrarlo definitivamente.

Sincronización de dispositivo

Sube fotos y videos directamente desde el navegador de cualquier dispositivo de tu red.

Indexación automática

Escaneo incremental con extracción EXIF, generación de miniaturas en tres tamaños y progreso en tiempo real.

PWA instalable

Instala Photonne como app nativa en móvil y escritorio. Carga offline del app shell incluida.

Multi-usuario

Roles y permisos por álbum y carpeta. Admin principal protegido e inquebrantable.

Notificaciones

Sistema de notificaciones para eventos del sistema, progreso de indexación y alertas de la biblioteca.

Administración completa

Panel de admin con gestión de usuarios, bibliotecas externas, copia de seguridad, detección de duplicados y archivos pesados.

Instalación

En marcha en tres pasos

Solo necesitas Docker. Photonne se encarga del resto.

1

Clona el repositorio

git clone https://github.com/marccafo/photonne.git
cd photonne
2

Configura el entorno

cp .env.example .env
# Edita .env: ruta de tus fotos, contraseñas y JWT key
3

Levanta los contenedores

docker compose up -d

Accede a http://tu-servidor:1107 e inicia sesión con el usuario admin configurado en .env.

Consulta el README completo para opciones avanzadas de configuración y despliegue.
